# Zero-Shot Learning

Zero-shot learning refers to a model's ability to perform a task or recognize a category that it has never seen explicit training examples for, by leveraging knowledge transferred from related tasks or categories seen during training. The "zero" in zero-shot refers to the absence of labeled training examples for the target task or class; the model must generalize using only its pre-trained representations and any task description provided at inference time.

In the context of large language models, zero-shot capability arises from the breadth of pre-training. A model trained on a vast corpus of text encounters numerous examples of reasoning, instruction following, question answering, summarization, and other tasks embedded within the text itself — in tutorials, textbooks, code, and online discussions. This exposure allows the model to recognize the structure of new tasks and apply relevant skills without explicit fine-tuning. GPT-3's 2020 paper, "Language Models are Few-Shot Learners," demonstrated remarkable zero-shot performance on standardized benchmarks, showing that scale and pre-training diversity can substitute for task-specific supervision in many settings.

Prompting is the primary mechanism through which zero-shot capability is elicited. By providing a natural language description of the task at the start of the model's context — sometimes called a zero-shot prompt or system instruction — the model is directed to interpret subsequent inputs in a particular way. For example, prepending "Translate the following text from English to French:" causes the model to treat the following text as a translation request, even without any demonstration examples.

Zero-shot generalization in the traditional machine learning sense — generalizing to unseen output classes in classification — relies on auxiliary information about the unseen classes, such as semantic attribute descriptions or word embeddings of class names. For image classification, a model might learn to recognize "zebra" as a class it has never seen during training by associating it with attributes like "striped," "hooves," and "similar to a horse" that it learned from seen classes. The model essentially maps new class descriptions into its learned semantic space and classifies test examples based on proximity to those descriptions.

Zero-shot performance degrades predictably as the target task diverges from anything encountered during pre-training. Tasks requiring specialized domain knowledge — such as diagnosing medical conditions from lab values or analyzing legal contracts under a specific jurisdiction's law — are unlikely to be well-represented in general pre-training corpora, limiting zero-shot capability even for large models. In such settings, at least a small number of examples (few-shot learning) or explicit fine-tuning is typically necessary.

CLIP (Contrastive Language-Image Pre-training) from OpenAI is a prominent example of zero-shot generalization extended to multi-modal settings. By training on image-text pairs with a contrastive objective, CLIP learns aligned visual and textual representations that allow it to classify images into arbitrary categories by comparing image embeddings with text embeddings of class descriptions — achieving competitive zero-shot image classification across many datasets without any labeled image training data.
