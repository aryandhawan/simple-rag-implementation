# Few-Shot Learning

Few-shot learning is the ability of a model to learn to perform a new task from a very small number of labeled examples — typically between one and a few dozen. This is in contrast to standard supervised learning, which requires hundreds to thousands of labeled examples per class to achieve acceptable performance. The goal of few-shot learning research is to build models that generalize from limited data the way humans do: a child who sees three examples of a giraffe can reliably recognize giraffes thereafter without needing thousands more.

In the context of large language models, few-shot learning takes a specific form called in-context learning. Rather than updating the model's parameters based on the examples, the examples are simply placed in the model's context window as part of the prompt. The model reads the (input, output) pairs as context and extrapolates the pattern to a new input appended at the end. GPT-3 demonstrated that models with sufficient scale can perform this in-context pattern matching with impressive reliability, achieving competitive performance on many NLP tasks using just a handful of examples.

The structure of a few-shot prompt is a concatenation of example demonstrations followed by the test input. For a sentiment classification task, a few-shot prompt might provide three examples: "Review: The movie was fantastic. Sentiment: Positive. Review: The food was cold and tasteless. Sentiment: Negative. Review: It was okay. Sentiment: Neutral." followed by the test review. The model, having seen the pattern, applies the same mapping to the new review. The number of examples is limited by the context window length.

In classical machine learning, few-shot learning is addressed through meta-learning — training a model explicitly to learn quickly from small datasets. The model is trained on many different few-shot tasks (each drawn from a large training distribution) so that it learns not specific class features but a general strategy for rapid adaptation. Prototypical Networks, MAML (Model-Agnostic Meta-Learning), and Matching Networks are established approaches in this tradition.

The sensitivity of in-context few-shot learning to the choice and ordering of examples is a well-documented phenomenon. The same model can show substantially different performance on the same test query depending on which demonstration examples are selected and in what order they appear. Examples that are more similar to the test instance tend to be more helpful, which has motivated k-nearest-neighbor retrieval of demonstrations from a large example pool — a technique called retrieval-augmented in-context learning or dynamic demonstration selection.

Few-shot learning is particularly valuable in domains where labeled data is scarce and expensive to obtain: medical imaging, rare language processing, legal document analysis, and specialized scientific domains. In these settings, a practitioner might obtain ten to fifty expert-labeled examples and use them as few-shot demonstrations or as fine-tuning data, rather than attempting to collect the thousands of examples that standard training would require.
