# Transformers

The Transformer is a neural network architecture introduced in the 2017 paper "Attention Is All You Need" by Vaswani et al. It replaced recurrent neural networks as the dominant architecture for sequence modeling tasks and has since become the foundation for virtually all large language models. The key innovation of the Transformer is the self-attention mechanism, which allows every position in a sequence to attend to every other position simultaneously, bypassing the sequential dependencies that made RNNs slow to train.

A standard Transformer is organized into an encoder and a decoder, each composed of stacked identical layers. The encoder reads the input sequence and produces a set of contextualized representations. The decoder generates the output sequence one token at a time, attending both to its own previously generated tokens and to the encoder's output via cross-attention. Models like BERT use only the encoder for classification and representation tasks, while GPT-style models use only the decoder for autoregressive text generation.

Each layer of the Transformer contains two main sub-components: a multi-head self-attention block and a position-wise feed-forward network. The self-attention block computes, for each token, a weighted combination of all other tokens' values, where the weights are determined by how relevant each other token is to the current one. The feed-forward network then applies a non-linear transformation independently to each position. Residual connections and layer normalization are applied around both sub-components to stabilize training.

Because the Transformer has no inherent notion of sequence order — attention is order-agnostic — positional encodings are added to the token embeddings before they enter the first layer. The original paper used fixed sinusoidal functions of position; later models use learned positional embeddings or more sophisticated schemes like rotary position embeddings (RoPE).

Scaling Transformers has proven remarkably effective. Increasing the number of layers, the size of the attention heads, and the dimensionality of the hidden states consistently improves performance on downstream tasks, following predictable power-law scaling curves. This empirical observation drove the development of models with tens of billions to hundreds of billions of parameters.

Computational efficiency is one of the main practical challenges with Transformers. Self-attention is quadratic in the sequence length: processing a sequence of n tokens requires computing n² attention scores. Various approximations — such as sparse attention, linear attention, and sliding-window attention — have been proposed to reduce this cost and allow Transformers to handle longer contexts efficiently.
